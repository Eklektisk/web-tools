Setting Up RSS

If you're reading this, then an RSS feed should be successfully set up on this page!

I was suprised how easy this was to set up. I was expecting to do some server-side scripting to get it set up, but it only took writing a single XML file. In hindsight, I have no idea why I thought this part would be harder. The hardest part was writing bash scripts to automate post generation from a HTML files, but that's just for ease of use making posts rather than for setting up RSS.
